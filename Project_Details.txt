## How to Run

1.  **Start the Data Simulation** (Terminal 1)
    ```bash
    python simulate_all.py
    ```
    *Leave this running to generate live data.*

2.  **Launch the Dashboard** (Terminal 2)
    ```bash
    streamlit run dashboard.py
    ```

3.  **Experience Live Mode**
    - Open the dashboard in your browser.
